# 🛸 Ben 10 Alien Web App

A modern and interactive **Ben 10 themed web application** built using React and Vite.
This project showcases dynamic UI, engaging design, and smooth performance for exploring alien characters.

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate to project folder
cd ben10-main/vite-project

# Install dependencies
npm install

# Start development server
npm run dev
```
